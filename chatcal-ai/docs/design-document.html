Executive Summary

ChatCal.ai is an AI-powered appointment booking assistant that integrates with Google Calendar. Built with Python, LlamaIndex, and Anthropic's Claude, it provides a friendly, conversational interface for scheduling meetings. The application is fully containerized for easy deployment across any platform.
